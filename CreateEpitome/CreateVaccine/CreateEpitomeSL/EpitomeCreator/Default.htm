About

The tool takes amino acid sequences (or text) as input and produces epitomes of all lengths.
 

Quick Help

Paste (cntl-V) or upload text input text or try a sample.

Start with "Compute Results" button.

Select all results (cntl-A)and copy (cntl-C).

To see all work steps, uncheck the box.
